\# SHL Project



This repository contains files for the SHL Assessment project.



\## Folder Structure



\- api/ : API code

\- llm/ : LLM parser scripts

\- shl\_assessments\_clean.csv : Input data

\- evaluation.py : Evaluation script

\- generate\_predictions.py : Prediction script

\- test\_predictions.csv : Test data

\- requirements.txt : Required Python packages
